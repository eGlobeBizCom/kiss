KIPR's Instructional Software System
====================================

Requirements
===========

* Qt 5.0 or higher
* CMake 2.8.12 or higher
* QScintilla2
* libkovanserial
* pcompiler
* libkar

Building (OS X and Linux)
=========================

	cd kiss
	mkdir build
	cd build
	cmake ..
	make

You should now have KISS ready to go in the deploy/ directory.
