KISS IDE 4.2 Tutorial: Hello World!

KISS IDE is an instructional integrated development environment for the C and C++ programming languages. In this tutorial, you will create and run a Hello World project.

Creating a New Project

First, you need to create a new project.

  1. Click the "New Project" icon in the center of the KISS IDE window. (Alternatively, you can go to File -> New -> New Project.)
  2. Enter a project name (e.g. "HelloWorld") and choose a save location (e.g. your "My Documents" folder). This information will be important in finding your project at a later time.
  3. Press "OK" to create the new project. The project sidebar will appear on the right hand side of the KISS IDE window, and your new project will be listed there.

Adding a New File

Next, you need to add a file to your project. This file will contain your program.

  1. After creating the project, a dialog will prompt you to add a new file to the project. Click "Yes." (Alternatively, you can right-click on the project in the sidebar and click "Add New File...").
  2. This will bring up the "Templates" dialog, which contains several sample programs. Select C Files -> Hello, World! and click "OK."
  3. Give the new file a name (e.g. 'hello.c') and click "OK." Your new file will automatically open in the KISS IDE editor.

Compiling and Running

Now that you've created a project and source file, it's time to compile and run your program!

  1. Click "Run" on the toolbar at the top of KISS IDE.
  2. Since this is the first time you've compiled or run this project, a "Target Selection" dialog will appear. You can run your program on a number of targets including your computer, a simulator, and a KIPR Link. For now, run the program locally on your computer by choosing the "My Computer" target and clicking "OK."
  3. You will see a progress indicator at the bottom of KISS IDE while your program compiles. Once compilation finishes, the progress indicator will disappear and your program will run inside the "My Computer" application. You can leave this "My Computer" application open if you plan on running more programs.
