KISS IDE 5.0 Tutorial: Existing Files

KISS IDE is an instructional integrated development environment for the C and C++ programming languages. In this tutorial, you will create a project that contains a file you already have.

Method 1: Adding an Existing File to a Project

The preferred method of using existing files is to directly add them to a project.

  1. Open a project by clicking on the "Open" icon in the center of the KISS IDE window and choosing the .kissproj file in your project folder. (Alternatively, you can go to File -> Open...) If you have not created a project yet, create one by clicking on the "New Project" icon. When prompted to add a new file, click "No."
  2. In the project sidebar, right-click on your project and click "Add Existing Files..." Choose the file(s) you want to add to your project.
  3. A dialog will ask you how to add the file(s) in the project. In most cases, one of the first two options is sufficient.
    1. Make a copy and move the copy to the project folder: Copies the original file into the project folder. The original file remains in the original location.
    2. Move to the project folder: Moves the original file into the project folder. The original file does not remain in the original location.
    3. Make a reference (absolute path): The project adds a reference to the file instead of adding it to the project folder. The original file is not touched. The reference is made using an absolute path, which is useful if the file never moves.
    4. Make a reference (relative path): Same as above, but the reference is made using a relative path, which is useful if the project and file move together.
    Choose an option and click "OK."
  4. The file(s) will appear under your project in the project sidebar. You may have to expand your project by double-clicking it.

Method 2: Creating a Project from a File

Alternatively, you can create a new project out of an existing file that is not already part of a KISS project.

  1. Open the existing file by clicking on the "Open" icon in the center of the KISS IDE window and choosing the file. (Alternatively, you can go to File -> Open...) You may have to change the filter from "KISS Project (*.kissproj)" to "All Files (*)".
  2. A dialog will prompt you to create a project with this file. Click "Yes."
  3. Give the project a name and save location. Click "OK."
  4. The new project, including the existing file, will appear in the project sidebar. You may have to expand your project by double-clicking it to see the file.

Note: The file is copied to the project folder, so the original file remains in the original location. See Method 1 to add more existing files.
